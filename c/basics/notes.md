
# C Language Notes

For C programming:
`PRINTF(3)`

| Control Sequence | Produces              |
|------------------|-----------------------|
|       `\n`       |    newline            |
|       `\l`       |    line-feed          |
|       `\r`       |    return             |
|       `\t`       |    tab                |
|       `\b`       |    backspace          |
|       `\f`       |    form-feed          |
|       `\s`       |    space              |
|  `\E` and `\e`   | escape character      |
|      `^x`        |`control-x` (`x`=char) |
